# Ruby configuration

Check ruby version
```bash
rbenv versions
```

Install new ruby version
```bash
rbenv install -v 2.6.6
rbenv global 2.6.6
```

## Install rubocop

Install rubocop gem
```bash
gem install rubocop
```

Verify installation
```bash
which rubocop
```
